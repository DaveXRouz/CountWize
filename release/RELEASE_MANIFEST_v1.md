## Netlify Configuration

### Publish Directory
```
publish = "site"
```

**Why**: The `/site` directory contains only deployable assets. This prevents internal documentation (baseline_phase_*, risk registers, runbooks) from being publicly accessible via Netlify.

### Key Redirects
- `www.countwize.com/*` → `countwize.com/:splat` (301, canonical domain)
- Clean URL rewrites for all main pages
- Custom 404 handler

### Headers
- Security headers (X-Frame-Options, X-XSS-Protection, CSP)
- Cache-Control for static assets (images, CSS, JS)
- HTML pages: 1-hour cache with must-revalidate
