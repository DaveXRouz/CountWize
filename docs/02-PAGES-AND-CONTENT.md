**20-Step Multi-Step Form:**
1. Start screen

3. Amount lost (radio)
4. Which cryptocurrency (radio)
5. When did it happen (radio)
6. Recipient wallet address (text)
7. Your wallet address (text)
8. Which wallet used (radio)
9. Where bought (radio)
10. Centralized vs Decentralized (radio)
11. Wallet access status (radio)
12. Contacted exchange? (radio)
13. Transaction proof (file upload, max 3)
14. Scam/fraud case? (radio)
15. Reported to authorities? (radio)
16. Urgency level (1-5 scale)
17. Own other crypto? (radio)
18. Open to security measures? (radio)
19. Country selection + Contact info (name, email, phone, consultation preference)
20. Schedule call (date/time picker, communication method)

**Success Screen:**
- Case ID: CW-[5-digit-number]
- "Our experts will contact you within 24 hours"
